# ORM-E-Commerce

## Description
This is a simple e-commerce application built using the Object-Relational Mapper (ORM) library. The application allows users to create, read, update, and delete products, as well as add and remove items from a shopping cart.

## Usage

To use this project, follow these steps:

1. Clone the repository: `git clone https://github.com/scottd988/ORM-E-Commerce
2. Install the dependencies: `npm install`
3. Run seeds file: `npm run seed`
4. Start the server: `npm run start`
5. Navigate to `localhost:3000` to view the application.
